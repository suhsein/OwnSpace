캘린더

    • 캘린더의 왼쪽 버튼을 누르면 이전달로, 오른쪽 버튼을 누르면 다음달로 이동합니다.
    • 캘린더의 각 날짜에는 해당일의 일정 갯수만큼 뱃지에 숫자가 표시됩니다.
    • 날짜를 클릭하면 아래 기능들을 사용할 수 있습니다.
      • 일정 등록
      • 일정 목록 보기

    일정 등록하기

    • 제목
      • 일정을 구분할 수 있도록 제목을 입력할 수 있습니다.
    • 장소
      • 장소를 직접 입력할 수 있습니다.
      • 지도를 열어서 장소를 검색할 수 있습니다.
      • 검색된 장소로부터 마커를 클릭하거나 장소 목록에서 장소를 클릭하면, 해당 장소명이 장소 입력칸에 입력됩니다.
    • 시간
      • 오전/오후의 시간을 스크롤하여 선택할 수 있습니다.
    • 상세 정보
      • 상세한 일정 정보를 설명 창에 입력할 수 있습니다.

    일정 목록 보기

    • 일정 보기
      • 등록한 일정의 제목, 장소, 시간, 상세설명을 볼 수 있습니다.
    • 일정 완료
      • 완료 버튼을 클릭하면 일정을 완료 처리할 수 있습니다.
      • 완료 상태에서 활성화 버튼을 누르면 다시 일정을 활성화 할 수 있습니다.
    • 일정 삭제
      • 삭제 버튼을 누르면 일정을 목록에서 제거할 수 있습니다.

    지도

    • 지도에 키워드를 입력합니다
      • 예시) 전주 미용실, 전주 맛집
    • 해당하는 장소들이 왼쪽 목록에 표시됩니다.
    • 목록의 장소명, 주소, 전화번호 클릭 시, 클립보드에 복사됩니다.
    • 목록의 장소 혹은 지도에 마킹된 장소 위에 커서를 두면 장소명을 볼 수 있습니다.

    갤러리

    • 목록에서 게시된 사진과 제목을 볼 수 있습니다.
    • 게시물의 첨부파일 목록 중 첫번째 첨부파일이 썸네일로 보여집니다.

    글쓰기

    • 글쓰기 버튼을 눌러서 글을 쓸 수 있습니다.
      • 게시물에는 무조건 하나 이상의 사진을 첨부해야합니다.
      • 사진 첨부 시 첨부 사진 목록이 표시됩니다.
      • 첨부 사진 목록에서 삭제 버튼을 눌러서 첨부한 사진을 삭제할 수 있습니다.

    • 게시물의 수정과 삭제가 가능합니다.
      • 게시물 수정 시에도 하나 이상의 사진이 남아있어야합니다.
      • 기존에 첨부했던 사진을 삭제하거나 새로운 사진을 첨부할 수 있습니다.
